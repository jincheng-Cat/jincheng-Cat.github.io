SM4

Posted on 2025-09-16 Edited on 2025-09-17 In crypto

分组/密钥：128 位 / 128 位，32 轮（不平衡 Feistel，SM4 风格）

概述

  • 结构：基于 32 位字的不平衡 Feistel
  • 字节序：标准大端（big-endian），16 字节装成 4 个 32 位字
  • 解密：与加密相同，轮密钥倒序使用即可

轮函数

轮函数定义为

T(x)=L(τ(x)) T(x) = L(\tau(x)) T(x)=L(τ(x))
  • τ\tauτ：对 32 位字的 4 个字节逐字节过 8→88\to 88→8 的 S 盒
  • LLL：线性扩散
L(x)=x⊕(x⋘2)⊕(x⋘10)⊕(x⋘18)⊕(x⋘24) L(x) = x \oplus (x\lll2) \oplus (x\lll10) \oplus (x\lll18) \oplus (x\lll24) L(x)=x⊕(x⋘2)⊕(x⋘10)⊕(x⋘18)⊕(x⋘24)

密钥扩展（Key Schedule）

  • 同一个 S 盒，线性层换成 L′L'L′：
L′(x)=x⊕(x⋘13)⊕(x⋘23) L'(x) = x \oplus (x\lll13) \oplus (x\lll23) L′(x)=x⊕(x⋘13)⊕(x⋘23)
  • 用 FK[4]FK[4]FK[4] 与 CK[32]CK[32]CK[32] 生成 32 个轮密钥 rk[0…31]\mathrm{rk}[0\ldots31]rk[0…31]。

加/解密与密钥扩展（Python 参考实现）

细节说明

  1. 每一轮先用 S 盒提供非线性（混淆），再用多次循环左移 + 异或提供线性扩散（扩散）。

  2. 轮函数 TTT 分为两步 τ\tauτ（S 盒）+ LLL（扩散）。

    • τ\tauτ：逐字节过 S 盒（对 32 位字 x=b0∥b1∥b2∥b3x=b_0\|b_1\|b_2\|b_3x=b0​∥b1​∥b2​∥b3​，把每个字节丢进 8→8 的 S 盒得到 τ(x)\tau(x)τ(x)）。
    • LLL：把一个 32 位字做若干循环左移后异或： L(y)=y⊕(y⋘2)⊕(y⋘10)⊕(y⋘18)⊕(y⋘24). L(y) = y \oplus (y\lll2) \oplus (y\lll10) \oplus (y\lll18) \oplus (y\lll24). L(y)=y⊕(y⋘2)⊕(y⋘10)⊕(y⋘18)⊕(y⋘24).

  3. 一轮流程重述：

    • step1: s=X[i+1]⊕X[i+2]⊕X[i+3]⊕rk[i]s = X[i+1] \oplus X[i+2] \oplus X[i+3] \oplus rk[i]s=X[i+1]⊕X[i+2]⊕X[i+3]⊕rk[i]
    • step2: t=T(s)t = T(s)t=T(s)（即先 τ\tauτ 再 LLL）
    • step3: X[i+4]=X[i]⊕tX[i+4] = X[i] \oplus tX[i+4]=X[i]⊕t
      32 轮后输出按反序拼接为 C=X[35]∥X[34]∥X[33]∥X[32]C = X[35]\|X[34]\|X[33]\|X[32]C=X[35]∥X[34]∥X[33]∥X[32]。

  4. 轮密钥来源：主密钥按大端拆成 MK0..MK3，与 FK[0..3] 异或得到 K0..K3，随后迭代生成 K4..K35（每步用 T′T'T′ 与 CK[i]），并把 Ki+4K_{i+4}Ki+4​ 作为 rk[i]rk[i]rk[i]。

设计目的与安全考量

  • S-盒（τ\tauτ）：提供非线性，阻断简单线性/差分传播。
  • LLL 与 L′L'L′（多次循环左移 + 异或）：提供快速扩散，字内和字间影响传播。
  • Feistel 架构：保证可逆性并允许复杂轮函数。
  • CK / FK：破坏对称性，降低相关密钥攻击风险。
  • 32 轮：通过多轮累积将局部非线性与简单扩散叠加为全局混合。
